# ZenHealth-AI

ZenHealth AI is an intelligent medical triage assistant designed to help users understand the urgency of their symptoms. By leveraging **FastAPI** and the **Gemini 2.5 Flash** model, the system analyzes user input, age, and medical history to provide structured, safe, and actionable health insights.

## 🚀 Features
* **Intelligent Triage:** Categorizes urgency into RED (Emergency), YELLOW (Urgent), or GREEN (Non-urgent).
* **Structured AI Analysis:** Uses Pydantic models to ensure AI responses are consistent and data-ready.
* **Patient-Centric Logic:** Factors in age and pre-existing conditions for a personalized assessment.
* **Modern API:** Built with FastAPI for high-speed, asynchronous request handling.

## 🛠️ Tech Stack
* **Backend:** Python, FastAPI, Uvicorn
* **AI Engine:** LangChain, Google Gemini 2.5 Flash
* **Frontend:** HTML5, CSS3, Vanilla JavaScript

## ⚖️ Ethical Disclosure
ZenHealth AI is an educational tool and a triage assistant. It is not a licensed medical professional. It is designed to assist in decision-making, not to provide a definitive diagnosis. In the event of a medical emergency, users are always instructed to contact local emergency services immediately.
